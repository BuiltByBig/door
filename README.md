# door

RFID doorlock application.


## Setup

1. Install nvm
2. `nvm install`
3. `nvm use`
4. `npm install`


## Development

- `npm run watch` runs dev server and tests.
- `npm test` runs test suite once.
- `npm run watch-test` runs test suite watching for changes.
